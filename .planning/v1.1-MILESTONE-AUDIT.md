## Reproduced Integration Failures

### 1. Event-field guard bindings fail

Reproduced with a temporary rules file containing:

```text
rule "guard.home_only" {
  priority 1
  lifetime permanent
  on TouchdownScored(team)
  when team == "home"
  before {
    emit AwardPoints(team: team, points: 1)
  }
}
```

Command:

```bash
HERDINGCATS_DSL_RULES_PATH=/tmp/event_guard.cats cargo build --manifest-path examples/dsl_consumer/Cargo.toml
```

Observed failure:

- `Validation: rule guard.home_only references unapproved binding team`

### 2. Duplicate emitted operation families break code generation

Reproduced with two rules that both emit `AwardPoints(...)`.

Command:

```bash
HERDINGCATS_DSL_RULES_PATH=/tmp/duplicate_emit.cats cargo build --manifest-path examples/dsl_consumer/Cargo.toml
```

Observed failure:

- generated consumer compile errors caused by duplicate generated variants / non-exhaustive matches after duplicate op emission
